# ⚡ HabitFlow: Daily Habit Tracker

A beautiful, modern, and fully responsive web-based habit tracker designed to help you build better routines, stay focused, and analyze your progress over time. 

Built entirely with standard web technologies (HTML, CSS, and Vanilla JavaScript), HabitFlow requires no complex backend or databases—all your data is securely stored right in your browser.

> Created by **Piyush Kumar**

---

## ✨ Features

* **📊 Interactive Dashboard:** Get a quick overview of your daily progress, active streaks, and remaining tasks for the day.
* **✅ Monthly Tracker Grid:** A visual, calendar-like view of your habit consistency throughout the month.
* **🎯 Habit Management:** Create custom habits with unique names, emojis, categories, and custom color tags.
* **⏱️ Built-in Focus Timer:** Includes a Pomodoro timer, short/long breaks, a custom timer, and a stopwatch. You can even link focus sessions to specific habits!
* **📈 Advanced Analytics:** Track your 30-day completion rates, current streaks, and all-time best streaks with visual progress bars.
* **🎨 Customization:** Personalize your experience by choosing your favorite UI accent color and setting daily goals.
* **💾 Data Ownership (Import/Export):** Your data stays on your device using `localStorage`. You can export your data as a JSON backup or CSV spreadsheet, and import JSON backups at any time.

---

## 🛠️ Tech Stack

This project is lightweight and fast, built without any external frameworks or libraries:
* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom properties (variables), modern Flexbox/Grid layouts, and fully responsive design optimized for desktop and mobile.
* **JavaScript (ES6+):** Vanilla JS for state management, DOM manipulation, timer logic, and local storage handling.
